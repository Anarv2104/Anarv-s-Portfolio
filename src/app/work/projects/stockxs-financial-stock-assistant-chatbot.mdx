## Overview

StockXs is a comprehensive financial analysis application built with Streamlit that combines AI-powered insights with real-time financial data. The app provides stock analysis, technical indicators, and interactive visualizations to support informed investment decision-making.

## Key Features

- **Real-Time Stock Data**: Fetches live stock prices and market data using Yahoo Finance API integration.
- **Technical Analysis**: Calculates key technical indicators including SMA (Simple Moving Average), EMA (Exponential Moving Average), RSI (Relative Strength Index), and MACD (Moving Average Convergence Divergence).
- **AI-Powered Insights**: Leverages OpenAI API to provide intelligent financial analysis and investment recommendations.
- **Interactive Visualizations**: Creates dynamic charts and graphs for stock price history and technical indicator trends.
- **User-Friendly Interface**: Built with Streamlit for an intuitive and responsive web application experience.
- **Comprehensive Analysis**: Combines quantitative data with AI-generated insights for holistic financial analysis.

## Technologies Used

- **Frontend**: Streamlit for interactive web application interface
- **AI Integration**: OpenAI API for intelligent financial analysis and recommendations
- **Data Source**: Yahoo Finance API for real-time stock market data
- **Data Analysis**: Python libraries (Pandas, NumPy) for financial calculations
- **Visualization**: Matplotlib/Plotly for interactive charts and graphs
- **Technical Indicators**: Custom algorithms for SMA, EMA, RSI, and MACD calculations
